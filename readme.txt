== Description ==

A rain of papier-mâché to show you have something to celebrate. This plugin is free software, and you can use it in commercial and non-commercial sites. This plugin uses __[confetti-js](https://github.com/Agezao/confetti-js "confetti-js")__ from __[Gabriel Age](https://github.com/Agezao "Agezao")__.

Screenshots are in spanish because it's my native language. As you should know yet I __spe'k__ english, and the plugin use it by default.

== Installation ==

1. Decompress papier-mache.zip and upload `/papier-mache/` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the __Plugins__ menu in WordPress.
3. Go to Appearance > Customize > Papier-mâché to configure the render of confetti rain.
